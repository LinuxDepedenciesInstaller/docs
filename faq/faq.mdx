## Frequently Asked Questions

<Accordion>

  <AccordionItem title="Do I need root or sudo?">
    No. LDI installs packages as a normal user. It is safe to run without root privileges.
  </AccordionItem>

  <AccordionItem title="Can I run LDI multiple times?">
    Yes, LDI is repeatable. Already installed packages will be skipped automatically.
  </AccordionItem>

  <AccordionItem title="Which packages are installed?">
    LDI installs the following essential tools:
    <ul>
      <li>curl</li>
      <li>wget</li>
      <li>tar</li>
      <li>zip / unzip</li>
      <li>git</li>
    </ul>
    Example to check installation:
    <CodeBlock>
      curl --version
      git --version
    </CodeBlock>
    <CopyButton content="curl --version\ngit --version" />
  </AccordionItem>

  <AccordionItem title="What distributions are supported?">
    LDI supports Debian-based Linux distributions, including Ubuntu, Debian, and derivatives.
  </AccordionItem>

  <AccordionItem title="How to update LDI?">
    Pull the latest version from GitHub:
    <CodeBlock>
      git pull origin main
    </CodeBlock>
    <CopyButton content="git pull origin main" />
  </AccordionItem>

  <AccordionItem title="Where can I get help?">
    Check the full documentation or open an issue on the GitHub repository:
    <a href="https://github.com/LinuxDepedenciesInstaller/LDI/issues" target="_blank">GitHub Issues</a>
  </AccordionItem>

</Accordion>
