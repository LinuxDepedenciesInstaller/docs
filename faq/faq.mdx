## Frequently Asked Questions

Get answers to common questions about installing and using LDI.

<Card
  title="Do I need root or sudo?"
  icon="shield"
  href="/getting-started/installation"
>
  No. LDI installs packages as a normal user. It is safe to run without root privileges.
</Card>

<Card
  title="Can I run LDI multiple times?"
  icon="repeat"
  href="/getting-started/usage"
>
  Yes, LDI is repeatable. Already installed packages will be skipped automatically.
</Card>

<Card
  title="Which packages are installed?"
  icon="boxes-packing"
  href="/getting-started/usage"
>
  LDI installs the following essential tools:
  <ul>
    <li>curl</li>
    <li>wget</li>
    <li>tar</li>
    <li>zip / unzip</li>
    <li>git</li>
  </ul>
</Card>

<Card
  title="What distributions are supported?"
  icon="desktop"
  href="/getting-started/installation"
>
  LDI supports Debian-based Linux distributions, including Ubuntu, Debian, and derivatives.
</Card>

<Card
  title="How to update LDI?"
  icon="sync"
  href="/getting-started/usage"
>
  Pull the latest version from GitHub:
  <pre><code>git pull origin main</code></pre>
</Card>

  title="Where can I get help?"
